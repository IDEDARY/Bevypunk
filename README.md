# Bevypunk

This repo is a recreation of ***Cyberpunk*** UI in ***Bevy*** using **[Bevy-Lunex](https://github.com/bytestring-net/bevy_lunex)** crate.

This project serves as advanced example on Bevy and Lunex workflow.
